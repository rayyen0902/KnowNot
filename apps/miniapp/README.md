# 知不Ai护肤小程序

这是独立可运行的 `Taro + React + TypeScript` 抖音小程序前端项目。

## 启动

在本目录执行：

```bash
pnpm install
pnpm dev:tt
```

## 构建

```bash
pnpm build:tt
```

## 目录说明

```text
apps/miniapp/
├── config/              # Taro 配置
├── src/                 # 页面、组件、数据
├── babel.config.js      # Babel 配置
├── package.json         # 运行脚本与依赖
└── tsconfig.json        # TypeScript 配置
```
