# 知不Ai护肤助理 - 交互逻辑与功能文档 ## 1. 页面跳转逻辑 | 出发页面 | 交互元素 | 目标页面 | 交互说明 | | :--- | :--- | :--- | :--- | | **首页 (Home)** | 左上角用户头像 | **个人中心** | 点击用户头像进入个人设置与数据概览 | | **首页 (Home)** | 底部滑动栏 "个人中心" Tab | **个人中心** | 快速切换至个人中心页面 | | **首页 (Home)** | 底部滑动栏 "护肤报告" Tab | **护肤报告** | 跳转至详细的肤质分析报告页 | | **首页 (Home)** | 底部滑动栏 "护肤日历" Tab | **护肤日历** | 跳转至每日护肤任务与排班页 | | **首页 (Home)** | AI消息中的 "点击查看报告" 气泡 | **护肤报告** | 直达生成的最新肤质报告 | | **所有二级页面** | 左上角返回箭头 | **首页** | 返回聊天主界面 | ## 2. 首页核心功能交互 (Main AI Chat) * **底部常驻栏**：输入框及上方的模式切换 Tab 栏（知心、顾问、专家等）始终固定在屏幕底部，不随对话内容滚动。 * **模式切换 (AI Personas)**： * **知心模式**：AI 话术更加感性、温暖，侧重于心理慰藉。 * **顾问模式**：AI 话术中立、专业，提供客观的护肤建议。 * **专家模式**：AI 话术硬核、严谨，多涉及成分分析与科学原理。 * **智能输入**：支持点击拍照（分析肤质）、语音消息（自然语言交互）及上传文件。 * **系统提示**：检测到用户在个人中心录入产品后，AI 在首页自动触发反馈：“检测到您已录入...，已为您加入今晚护肤日程。” ## 3. 护肤报告功能 (Skin Report) * **多维指标**：水分、油分、平滑度、敏感度四个维度的进度条动态展示。 * **成分建议**：基于肤质分析，明确标注“推荐成分”与“规避成分”。 * **早晚方案**：将护肤流程拆分为 AM 和 PM，并提供具体的操作指导。 ## 4. 护肤日历功能 (Skin Calendar) * **月度成就**：日历通过浅绿至翠绿的色阶展示每日护肤任务的达成度（翠绿为 100% 完成）。 * **任务打卡**：用户完成 AM 或 PM 的护肤流程后，点击末尾的勾选框进行状态同步。 ## 5. 个人中心功能 (Personal Center) * **我的护肤品 (核心数据源)**：用户在此录入自有产品，系统自动解析成分并同步至首页 AI 建议及护肤日历排班。 * **肤质变化曲线**：展示用户近期的肤质改善趋势，提供视觉化的成就感。 ## 6. 技术交付建议 (For Cursor/Developer) * **前端框架**：建议使用原生 HTML/CSS 结合轻量级 JS 或 React/Vue。 * **状态管理**：建议使用全局状态管理用户录入的护肤品数据，以便实时更新日历和首页 AI 话术。 * **字体**：全局强制使用系统默认字体 (`font-family: system-ui, -apple-system, sans-serif`)。
